# Chost Host
Chost Host is a responsive landing page for a hosting company, featuring a multi-section marketing layout with domain search UI, pricing plans, team presentation, testimonials slider, and service highlights. Built with semantic HTML, modern CSS (Flexbox + Grid), and a small amount of vanilla JavaScript for interactive UI elements.

## Live Demo
- https://valentyn-m.github.io/site-study-ChostHost/

## Key Features
- Multi-section landing page structure (Domain, Hosting/Packages, Clients, Blog, Support) with clear top navigation
- Domain search block UI (TLD selector + pricing cards)
- Pricing “Packages” cards with plan comparison and call-to-action buttons
- “Authorization” area with Login / Register CTAs
- Team/About section with profile cards
- Testimonials “Slider” section
- Additional services grid (Fast Servers / Cloud Servers / Domain Transfer / Live Support)
- Multi-column footer with navigation blocks and contact info

## Technologies Used
- HTML5 (semantic layout)
- CSS3
- Flexbox (layout)
- CSS Grid (layout)
- Vanilla JavaScript (basic interactivity)
- GitHub Pages (deployment)
